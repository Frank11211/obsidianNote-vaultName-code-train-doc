# Introduction Node.js
- Runtime environment 
- allows to run JavaScript on the server ( outside of web browser )
- single-threaded , but use an Event Loop & Non-blocking I/O
- Able handle thousands of concurrent connections (like a chat app or a food ordering system) 
- without stuck - wating for a database to respond. 

# What should u learn 
- Since you experience with API testing and C#, can focus on these core pillars 
	- Asynchronous Programming :-
		- "Soul" of Node.Js, you need master
			- ``Callbacks``, ``Promises`` , and ``Async/Await``
	- The Event Loop 
		- Understand how Node handles tasks without blocking 
	- NPM ( Node Package Manager )
		- world largest software registry 
		- learn how to manage dependencies and use package 
		( ``Express`` for APIs or ``Mongoose`` for database ) is essential 
	- Express.js 
		- Most popular framework for building web servers in Node.
	- Middleware 
		- learn how to process request ( checking if a user is logged in) before reaches your final logic.
