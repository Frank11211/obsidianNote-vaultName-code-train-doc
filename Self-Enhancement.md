
<h3> Communication Issue</h3>
- Can be solve by ***sense of logic*** + ***Practice***
- Factor cause message unclear 
	- Brain overloaded with info
	- Point unable pinpoint out

***Sense of Logic*** core : Conclusion first n Daily work conversation, follow the ***PREP*** principle :-

<table>
		<tr>
			<th>Command </th>
			<td>Remark</td>
		</tr>
		<tr>
			<td>P ( Point ) 结论</td>
			<td>State your core argument in your very first sentence</td>
		</tr>
		<tr>
			<td>R ( Reason ) 理由</td>
			<td>What led you to this conclusion ? ( 2-3 reasons ) </td>
		</tr>
		<tr>
			<td>E ( Example ) 案例</td>
			<td>State example or data to support your reason</td>
		</tr>
		<tr>
			<td>P ( Point ) 重申结论</td>
			<td>Finally, wrap things up &amp reinforce your point</td>
		</tr>
	</table>


***Example***
- Hi Koay CN, just a heads-up regarding meeting, the route from Greenland to Penang bridge is heavy traffic compare to usual due to accident. I barely made it on time by using shortcut but Lam will be caught in traffic since he live further up. I can confirm that traffic is effected everyone on that route today.
	PREP :-
	- P : colleague might late for meeting 
	- R : traffic jam ( cause by car accident)
	- E : using another route, but Lam still caught in traffic due to live further up

***Rule of Three*** 
	- Analysis and split into ***3*** point ( Method / Example / Effect )
		- Method - even you had only 2 point, force squeeze into 3 point 
		- Example - "About this project, 3 point I like to emphasize"
		- Effect - sound organize and logic
	
***Hedging & Softening ( 说话软一点)***
	-    This is impossible ❌
		Based on the current timeline, this might be challenge ✔️
	-    You are wrong ❌
		I have slightly different perspective on this ✔️
		
***Action-Oriented Closing ( 动作导向技术 )***
	- A good message should always end with an "Action".
	- Example:- 
		- "Let me know if you are okay with this."（征求意见）
		- "I will follow up by 5 PM."（给出承诺）
		- "Would you like me to `<Do Somethign>` ?"（主动提供帮助）
